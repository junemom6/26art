# 🌿 디지털환경전시회 (Digital Environment Exhibition)

> **"미래형 친환경 디지털 미술관"**  
> 초·중학생들이 직접 제작한 환경 관련 JavaScript 웹페이지 작품을 전시, 실행하고 피드백을 주고받는 현대적인 교육용 전시 플랫폼입니다.

---

## 📂 프로젝트 구조 (Architecture)

```text
digital-environment-exhibition/
│
├── index.html            # 전시회 메인 HTML 레이아웃 (전시관, 업로드, 배지, 관리자 화면)
├── styles.css            # 유리 모티브(Glassmorphism) 스타일 및 반응형 레이아웃, 애니메이션
├── app.js                # 로컬 DB(localStorage) 연동, 검색/필터링, 투어, 업로드, 배지 핵심 로직
├── mockData.js           # 초기에 탑재되는 학생 작품(6개) 및 배지 시스템 기본 정의 데이터
├── run_exhibition.bat    # Windows용 서버 원클릭 가동 파일 (Python 서버 실행)
│
└── projects/             # 실제 실행 가능한 학생 환경 작품들 (Iframe용 단독 게임/콘텐츠)
    ├── save-the-whale/       # 🐋 바다 고래를 구해줘! (쓰레기 클릭 수거 게임)
    ├── recycle-sorting/      # ♻️ 분리배출 마스터! (드래그 앤 드롭 분류 게임)
    ├── carbon-calculator/    # 🌱 나의 일일 탄소 발자국 계산기 (대화형 설문식 계산기)
    └── endangered-animals/   # 🐼 멸종위기 동물 수호대 도감 (인터랙티브 정보 카드)
```

---

## 🚀 실행 방법 (How to Run)

### 1. Windows 원클릭 실행 (추천)
1. `digital-environment-exhibition` 폴더 안의 **`run_exhibition.bat`** 파일을 더블클릭합니다.
2. 로컬 웹 서버가 가동되며 자동으로 기본 브라우저에서 `http://localhost:8000` 주소가 열립니다.

### 2. 수동 실행 (터미널)
프로젝트 폴더 경로에서 아래 명령어를 실행한 후 브라우저에서 `http://localhost:8000`에 접속합니다:
```bash
python -m http.server 8000
```

## ✨ 구현된 핵심 기능 (Key Features)

1. **친환경 디지털 미술관 비주얼 (Hero & Background Animation)**
   - 숲, 나뭇잎, 구름, 바다 파도가 부드럽게 움직이는 CSS 애니메이션 배경이 적용된 대형 Hero 영역.
   - 글래스모피즘(Glassmorphism)과 마이크로 인터랙션을 반영한 세련되고 현대적인 카드 UI.

2. **반응형 갤러리 그리드**
   - PC(4열), 태블릿(2열), 모바일(1열)로 레이아웃이 자동 조정되어 모든 기기 완벽 지원.
   - 카테고리 실시간 카운트 뱃지 및 사이드바 선택 필터링 기능.

3. **인터랙티브 작품 실행 모달 (Iframe Sandbox)**
   - 작품 카드 클릭 시, 모달창 내부 `iframe`을 통해 실제 학생들이 개발한 게임이 로컬 상에서 직접 작동합니다.
   - **전체화면 감상**, **새 창에서 열기**, **QR코드 공유** 지원.

4. **실시간 검색 및 환경 카테고리 필터**
   - 상단 검색창에 입력 즉시 제목, 학생 이름, 설명글에서 결과를 찾아 실시간으로 전시 카드가 갱신됩니다.
   - 사이드바 카테고리를 누르면 해당 작품만 깔끔하게 모아볼 수 있습니다.

5. **친구들의 피드백 & 좋아요 시스템 (실시간 LocalStorage 연동)**
   - 댓글 작성(이름 입력 및 **익명 작성** 선택 가능), 개별 댓글 추천(좋아요), 댓글 **수정 및 삭제(본인)** 가능.
   - 작품 자체의 좋아요 버튼 클릭 시 숫자가 1씩 증가하며 인기 순위에 즉시 반영됩니다.

6. **인기 작품 & 오늘의 추천작 전시**
   - 누적 좋아요 기준 실시간 **Top 5 랭킹** 보드 출력.
   - 상단에 강조 표시되는 **오늘의 추천 작품** 배너 탑재.

7. **간편한 드래그 앤 드롭 업로드 & 성공 애니메이션**
   - 웹사이트 파일(`.html`, `.css`, `.js`, `.zip`) 또는 폴더를 드롭존에 놓으면 로딩 게이지가 차오릅니다.
   - 업로드 성공 시, 캔버스를 수놓는 **화려한 컬러 축하 파티클(폭죽) 애니메이션** 발동.
   - 업로드한 작품은 정보 기입 후 관리자의 승인을 거쳐 전시관에 정식 등록됩니다.

8. **도슨트 자동 슬라이드 전시 모드 (Exhibition Tour)**
   - 미술관 키오스크처럼 10초마다 다음 작품으로 자동 이동하며 감상하는 기능.
   - 상단 진행률 프로그레스 바 제공 및 이전/다음/투어 종료 컨트롤 패널 지원.

9. **환경 배지 시스템 (Badge Checklist)**
   - 첫 방문(`초록빛 첫걸음` 🌱), 댓글 3개 작성(`소통의 새싹` 💬), 작품 업로드(`에코 크리에이터` 🎨), 좋아요 10개 달성(`인기 환경 작가` ⭐) 등 유저 활동에 따라 배지가 실시간으로 활성화(잠금 해제)되며 축하 토스트 팝업이 나타납니다.

10. **관리자 페이지 (School Admin Control)**
    - 승인 대기 작품을 검수하여 승인(`승인완료`) 또는 보류 처리 가능.
    - 부적절한 작품/댓글 삭제 권한 및 전체 학생/작품 검색 가능.
    - 카테고리별 전시 현황을 나타내는 **데이터 시각화 그래프(차트)** 제공.

11. **다크 모드 (Dark Mode)**
    - 야간이나 어두운 교실에서도 눈이 편안한 고급 다크 그린/블루 테마 지원.

## 🎨 기술 스택 및 디자인 정책
- **Frontend:** HTML5, CSS3 (Vanilla CSS variables), JavaScript (ES6+, LocalStorage state)
- **Design Concept:** 자연(Green #2E8B57)과 기술(Sky Blue #7EC8E3)의 조화, Glassmorphism, 둥근 모서리, 소프트 섀도우, 모바일 반응형 완벽 보장.
